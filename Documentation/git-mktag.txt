git-mktag(1)
============
v0.1, May 2005

NAME
----
git-mktag - Creates a tag object


SYNOPSIS
--------
'git-mktag'

DESCRIPTION
-----------
Reads a tag contents from its standard input and creates a tag object.
The input must be a well formed tag object.
